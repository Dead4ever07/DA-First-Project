Validates user input and computes the appropriate route. 

This function checks if the given inputs exist in the graph. Then, it determines the appropriate route computation based on the provided parameters. If the mode is "{}driving"{} with no constraints, it computes the best and alternative driving routes. Otherwise, it calculates a restricted or a driving-\/walking route.

{\em g} & Pointer to the graph. \\
\hline
{\em origin} & The ID of the source vertex. \\
\hline
{\em dest} & The ID of the destination vertex. \\
\hline
{\em vertex} & A vector of node IDs that should be avoided. \\
\hline
{\em edges} & A vector of pairs representing edges to be avoided. \\
\hline
{\em middle} & The ID of a mandatory intermediate vertex (-\/1 if not required). \\
\hline
{\em mode} & The selected travel mode ("{}driving"{} or "{}driving-\/walking"{}). \\
\hline
{\em max\+Walk\+Time} & The maximum allowed walking time for mixed routes. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
A formatted string containing the computed route or an error message if the input is invalid.
\end{DoxyReturn}
\begin{DoxyNote}{Note}

V is the number of vertices and E is the number of edges.
